# Description

This module adds checklists to leads. Configure your own checklist items via the crm configuration. All configured checklist items will automatically appear under a new "Checklist" tab within your leads.


# Installation

No specials, plain install. CRM needs to be available on your system.

# Usage / Context

With every lead you may have to follow repetetive steps. With this module you can define them, get the automatically added as a checklist to your lead and then track an overall progress of the checklist items done per lead.

# Configure

Go to CRM -> Configuration -> Checklist items

Here you can add multiple checklist items which will appear on every lead created from now on.

# Usage

Go the the "Checklist" page in your lead form view and check the checkboxes.
